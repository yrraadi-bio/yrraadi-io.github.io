# Advisor Images

Place advisor headshot images in this directory with the following names:

- `kellis.jpg` - Dr. Manolis Kellis
- `paulk.jpg` - Dr. Nicole Paulk  
- `bashir.jpg` - Dr. Rashid Bashir

**Image specifications:**
- Format: JPG or PNG
- Recommended size: 300x300px or larger (will be displayed as 120x120px circle)
- Square aspect ratio works best
- Professional headshot photos

The images will be displayed as circular avatars in the bio popup.
